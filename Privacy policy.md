# Политика конфиденциальности для приложения **SOTKA**

## 1. Введение

Данное приложение основано на оригинальной программе "SOTKA", созданной для развития физической культуры и здорового образа жизни. Подробную информацию о принципах конфиденциальности оригинальной программы можно найти на [сайте проекта](https://100.workout.su/privacy.html).

## 2. Какие данные мы собираем?

- Приложение следует принципу минимального сбора данных и собирает только информацию, необходимую для его функционирования: данные профиля и тренировок.
- Данные пользователя хранятся локально на устройстве и передаются на сервер при наличии подключения к интернету.
- Для диагностики ошибок и улучшения стабильности приложения также могут автоматически обрабатываться технические и аналитические данные с помощью сервисов Firebase Crashlytics и Firebase Analytics, включая сведения о сбоях, техническую информацию об устройстве и операционной системе, версию приложения, а также события использования интерфейса, такие как просмотры экранов и нажатия на кнопки.

## 3. Как используются ваши данные?

### 3.1 Основное использование

Ваши данные используются исключительно для:

- Обеспечения работы приложения на вашем устройстве
- Синхронизации данных между вашими устройствами (через iCloud)
- Отображения вашего прогресса и статистики тренировок
- Персонализации интерфейса согласно вашим настройкам
- Диагностики технических ошибок, анализа сбоев и улучшения стабильности приложения

### 3.2 Синхронизация с сервером

- Синхронизация происходит только при наличии активного интернет-соединения и только для авторизованных пользователей
- Данные передаются на сервер проекта SOTKA через защищенный протокол HTTPS
- Аутентификация осуществляется с помощью токена, закодированного в Base64
- Серверная синхронизация позволяет обмениваться информацией между вашими устройствами
- Данные неавторизованных пользователей не передаются на сервер

### 3.3 Firebase Crashlytics и Firebase Analytics

- Для диагностики ошибок и анализа сбоев приложение использует **Firebase Crashlytics**
- Для получения технической аналитики использования приложения используется **Firebase Analytics**
- Эти сервисы могут обрабатывать сведения о сбоях, технические данные устройства и операционной системы, версию приложения, а также события использования интерфейса, такие как просмотры экранов и нажатия на кнопки
- Эти данные используются только для диагностики проблем, улучшения стабильности и повышения удобства использования приложения
- Данные не используются для таргетированной рекламы

### 3.4 Отсутствие передачи третьим лицам для коммерческих целей

- Данные не передаются третьим лицам для коммерческих целей
- Нет продажи или обмена данными с рекламными сетями
- Нет использования данных для таргетированной рекламы
- Передача данных сторонним сервисам возможна только в объёме, необходимом для работы функций диагностики ошибок и технической аналитики приложения

## 4. Доступ к данным и безопасность

- Доступ к данным возможен только после аутентификации на устройстве
- Доступ к данным имеют только вы и лица, использующие ваше устройство
- Вы можете просматривать, изменять и удалять ваши данные из приложения
- Данные могут синхронизироваться между вашими устройствами через iCloud

## 5. Работа в офлайн-режиме

- Полноценная работа без подключения к интернету
- Локальное сохранение всех необходимых данных
- Синхронизация происходит только при наличии соединения
