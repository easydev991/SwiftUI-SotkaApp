## Основное приложение

### Главный экран

- Номер текущего дня
- Инфопост для текущего дня
- Выбор активности для текущего дня и переход к тренировке (если выбрана тренировка)
- Переход к внесению прогресса, если прогресс для текущего блока программы заполнен не целиком
- Переход ко списку всех инфопостов

#### Список инфопостов

Список всех инфопостов с возможностью добавлять в избранное и отметить пост прочитанным (жестами свайпа)

### Экран профиля

- Отображение профиля пользователя (аватар, логин, пол/возраст, страна/город)
- Редактирование профиля (изменение данных, аватара, смена пароля)
- Переход к дневнику тренировок
- Переход к экрану прогресса
- Переход к пользовательским упражнениям
- Выход из аккаунта с подтверждением
- Обновление профиля через pull-to-refresh

#### Редактирование профиля

- Изменение логина, email, полного имени
- Загрузка и изменение аватара (камера или галерея)
- Выбор пола, даты рождения
- Выбор страны и города из списка
- Смена пароля через отдельный экран

#### Дневник тренировок

- Табличное представление активностей с группировкой по секциям программы
  - Сортировка записей (прямая/обратная)
- Сеточное представление активностей с цветовой индикацией типа
- Переключение между видами отображения
- Просмотр и редактирование записей тренировок
- Добавление комментариев к активностям
- Удаление активностей с подтверждением

#### Прогресс

- Статистика прогресса с визуализацией 100 дней программы
- Отображение данных для контрольных точек (день 1, 49, 100)
- Редактирование метрик прогресса (подтягивания, отжимания, приседания, вес)
- Загрузка и редактирование фотографий прогресса (спереди, сзади, сбоку)
- Удаление прогресса с подтверждением
- Синхронизация с сервером

#### Пользовательские упражнения

- Список пользовательских упражнений с поиском и фильтрацией
- Создание кастомных упражнений с выбором иконки
- Редактирование и удаление упражнений
- Синхронизация с сервером

### Экран "Ещё"

- Настройки темы приложения (светлая/темная/системная)
- Настройки языка приложения (переход в системные настройки)
- Настройки тренировок (уведомления, время отдыха, звук таймера, вибрация)
- Сброс программы с подтверждением (доступен с дня 2)
- Оценка приложения в App Store
- Отправка обратной связи
- Поделиться приложением
- Просмотр инфопоста "О программе"
- и т.д.

#### Журнал синхронизаций

- Список всех синхронизаций, сгруппированных по дате
- Детальный просмотр результатов синхронизации (статистика, ошибки)
- Удаление всех записей журнала

### Экран авторизации

- Приветственный экран
- Вход через email и пароль
- Восстановление пароля

### Экран тренировки

- Превью тренировки с отображением упражнений дня
- Редактирование количества повторений и кругов/подходов
- Выбор типа выполнения (круги/подходы) для дней > 49 и "турбо" для дней 92-98
- Настройка времени отдыха между подходами/кругами
- Добавление комментария к тренировке
- Сохранение тренировки как выполненной
- Переход к выполнению тренировки
- Редактор упражнений с drag-and-drop для изменения порядка

#### Выполнение тренировки

- Пошаговое выполнение тренировки (разминка, упражнения, заминка)
- Таймер отдыха между этапами с круговой анимацией
- Прерывание тренировки с подтверждением
- Отслеживание времени тренировки
- Уведомления о завершении отдыха

## Приложение для часов

- Приложение на часах работает только после авторизации в основном приложении на iPhone
- Все данные запрашиваются с iPhone через WatchConnectivity в реальном времени, локальное хранение на часах не используется
- iPhone должен быть неподалеку и подключен к этим часам, чтобы на них работало приложение

### Главный экран

- Отображение текущего дня программы
- Выбор типа активности для текущего дня (тренировка, растяжка, отдых, болезнь)
- Отображение выбранной активности с данными тренировки (если выбрана тренировка)
- Изменение/удаление активности
- Переход к превью тренировки (если выбрана тренировка)
- Экран авторизации при отсутствии связи с iPhone

### Экран превью тренировки

- Отображение упражнений дня
- Выбор типа выполнения (круги/подходы) для базового и продвинутого блоков, и "турбо" для дней 92-98
- Редактирование количества повторений и кругов/подходов
- Редактирование упражнений с изменением порядка
- Сохранение тренировки как выполненной
- Переход к выполнению тренировки

### Выполнение тренировки

- Пошаговое выполнение тренировки (разминка, упражнения, заминка)
- Таймер отдыха между этапами тренировки
- Прерывание тренировки с подтверждением
- Автоматическое сохранение результата на iPhone (через WatchConnectivity)

### Синхронизация с iPhone

- Синхронизация статуса авторизации и текущего дня
- Синхронизация выбранных активностей
- Синхронизация результатов тренировок
- Синхронизация времени отдыха между подходами/кругами
- Обработка ошибок при отсутствии связи
